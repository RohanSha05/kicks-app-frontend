# Kicks App Frontend

## Overview

Kicks App Frontend is a Next.js storefront UI for browsing sneakers, viewing product details, and exploring featured items. It uses client-side data fetching and a modular component structure to keep the UI reusable and easy to evolve.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Axios

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the app:

```
http://localhost:3000
```

## Notes

- Product data is fetched from the EscuelaJS API.
- Static product images in the details page are placeholder assets under `public/assets/image`.
- Environment variables are not required for local development.
